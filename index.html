A Soldier In Action

Description

This is a mini platform shooter game based in Python using PyGame

Controls

    W: Jump

    A: Move Left

    D: Move Right

    Q: Throw Grenades

    Spacebar: Shoot Bullets

Features

There are pick up boxes for ammo, health and grenades which are used when you run into them
