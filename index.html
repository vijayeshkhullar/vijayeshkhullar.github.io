Gesture Puppet

This is an in progress Hand Gesture control project. It uses computer vision to detect hand gestures through the camera webcam. The detected gestures are sent to a microcontroller in which I am puppeteering a 3D printed hand that I am designing. This project can be implemented to puppeteer anything. I am using a GitHub repository originally from Kazuhito Takahashi, and translated by Nikita Kiselov which consists of the base code that maps and reads the hand gestures. I will provide the altered code and arduino code on my GitHub.

VAN HELSING

This is a platformer game about Van Helsing the Vampire Hunter who hunts vampires with guns. He also can throw magic crystals that explode. He must traverse through the levels and reach the exit in order to advance. It is based in Python using PyGame. The downloadables to play the game are on the right, however, they only work on windows. The controls are listed below:

    W: Jump

    A: Move Left

    D: Move Right

    Q: Throw Crystals

    Spacebar: Shoot Bullets

Click Here to download the Game and Here for the assets (download both then drag the game executable into the assets folder)

Automated Teller Machine

This is banking site using Java. It includes account registration, account login, deposits, withdrawals, transfers, account deletion and some other features. All that is needed is to connect to a server in order to keep accounts registered. This uses a GUI in eclipse and I have not tested it elsewhere.
