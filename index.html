A Soldier In Action

This is a mini-platform shooter involving one player, the green soldier, and red enemy soldiers. The main character can throw grenades and shoot bullets as well as pickup ammo and health boxes. He must traverse through the levels and reach the exit in order to advance The controls are listed below. It is based in Python using PyGame.

    W: Jump

    A: Move Left

    D: Move Right

    Q: Throw Grenades

    Spacebar: Shoot Bullets

Hangman

This is a text-based hangman game created using javascript. It randomly picks a word from an array, and the user is meant to input guesses till they run out of attempts. They can choose to play again, which loops to the beginning, or end the game, which ends the program. If you choose to play again, it keeps track of the wins and losses as well as the number of games until you end the program, which resets the count. It runs on the console in eclipse, and I have not tested it elsewhere.
